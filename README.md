# Agent IA de correction de copies écrites – NSI

## Objectif
Ce projet vise à créer un agent d'aide à la correction de copies écrites
en spécialité NSI. L'agent analyse les réponses des élèves à l'aide de mots-clés
et propose une note ainsi qu'un commentaire automatique.

## Principe
- Analyse du texte de la réponse
- Recherche de mots-clés importants
- Calcul d'un score
- Génération d'une note et d'un commentaire

## Limites
- Ne comprend pas les synonymes
- Peut mal interpréter des réponses originales
- Ne remplace pas un correcteur humain

## Technologies
- Python
- Système à règles
- Apprentissage par ajustement des critères
